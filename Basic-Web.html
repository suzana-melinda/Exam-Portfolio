MoodBoard

Moodboard of the given style

The first assignment from the Basic Web theme was to create a Moodboard of the given style (Art Deco).

In order to create it, a research was made to find what characterizes the style.

The moodboard was created in Adobe Photoshop using the basic tools such as the selection, crop and resize tools structured on layers.

I really enjoyed working on this assignment since Art Deco is one of my favorite styles.

Click here for the file

The process

The next phase of the project was to make a further research on the subject and gather content for the website such as articles, images and examples. Another task was to create a logo.

For the logo Adobe Illustrator was used. It was the first time when I have used it. At the beginning I had some trouble with it but with time I got better at it.

After gathering all the content the next task was to create my very first mobile website, using basic knowledge of HTML and CSS.

During this theme I have also learn some design principles and have been introduces to Adobe color wheel. Even though I had some knowledge of color and basic coding this was new information to me and it changed my perspective.

Prototype and Usability test

After finalizing the first web page another assignment was to make an interactive prototype in Adobe XD. It was my first time interacting with this program, it definitely helped to save time and test it.

Another useful tool that I have learned was the CSS Grid layout. Until then I was not familiar with it.

The Testing phase consisted in performing the “Trunk Test” in order to have a better insight and to make it more user friendly.

Final Results

Based on the results of the test I have made some changes such as making sure the user knows on witch page has landed by highlighting the navigation bar, some of the sections have been reorganized and started the coding process.

For the home and design pages, the css grid it was used.
